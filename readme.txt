== Description ==

**Mega Menu Builder for Elementor** is a powerful and feature-rich mega menu widget that extends Elementor's capabilities with advanced navigation options. Create stunning, fully responsive mega menus with ease - no coding required!

= Key Features =

**🎨 Multiple Layout Options**
* Horizontal navigation bar
* Vertical sidebar menu
* Mobile-responsive hamburger menu
* Customizable breakpoints

**📋 Dropdown Types**
* Simple list dropdowns
* Multi-column mega panels
* WordPress custom menu integration
* Recent posts display
* WooCommerce products showcase

**🎯 Advanced Customization**
* Unlimited menu items with icons
* Badge support (NEW, HOT, SALE)
* Custom colors and typography
* Hover animations (Slide, Fade, Zoom)
* Border, shadow, and spacing controls

**📱 Mobile Optimized**
* Responsive hamburger button

**🛒 WooCommerce Ready**
* Display products in dropdowns

* Filter by featured products
* Multiple sorting options

**✨ Posts Integration**
* Show recent posts in dropdowns

* Thumbnail and date display

**🎨 Style Controls**

* Spacing and alignment
* Icon customization

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/mega-menu-builder-for-elementor/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure Elementor is installed and activated
4. Edit any page with Elementor
5. Search for "Mega Menu" in the widgets panel
6. Drag and drop the widget onto your page
7. Customize your menu using the widget settings

== Frequently Asked Questions ==

= Do I need Elementor Pro? =

No, this plugin works with the free version of Elementor. However, it's fully compatible with Elementor Pro if you have it.

= Can I use WordPress menus? =

Yes! The widget includes a "WP Custom Menu" dropdown type that lets you display any WordPress menu you've created.

= Does it work with WooCommerce? =

Absolutely! You can display WooCommerce products in dropdown panels with prices, thumbnails, and sorting options.

= Is it mobile responsive? =

Yes, the menu is fully responsive with a hamburger button for mobile devices. You can customize the mobile breakpoint.

= Can I customize colors and fonts? =

Yes, you have complete control over colors, typography, spacing, borders, shadows, and more through Elementor's style controls.

= How many menu items can I add? =

There's no limit! Add as many menu items as you need with unlimited dropdown levels.

= Does it support RTL languages? =

Yes, the menu works perfectly with RTL (Right-to-Left) languages.

= Can I add icons to menu items? =

Yes, you can add icons from Font Awesome or upload custom SVG icons for any menu item.
